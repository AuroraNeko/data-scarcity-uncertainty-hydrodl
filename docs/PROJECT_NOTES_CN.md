## 一、论文信息

**暂定标题：** How Data Scarcity Compromises Uncertainty Estimates in Hydrological Deep Learning — and How to Fix It

**核心贡献：** 首次系统量化训练数据量对深度学习径流预测不确定性可靠性的影响，揭示数据稀缺导致的分位回归欠覆盖偏差，并验证 Conformalized Quantile Regression (CQR) 作为轻量修复方案的有效性。

## 七、关键教训

- **读完整论文。** MAML 教训——不读方法学就下结论严重误导
- **物理约束在归一化空间语义扭曲。** 非负惩罚在 log1p+归一化空间反而有害
- **所有 DL 模型过拟合。** Early stopping 必须有
- **Pinball Loss 比 MSE 更稳定。** 30 epoch 未过拟合
- **后台输出缓冲导致看不见训练日志。** py 脚本直接写文件更可靠
- **外部调研 + 读全文是必要的质量保障。** 两次外部调研确认了新颖性和贡献度
